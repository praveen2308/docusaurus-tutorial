System Requirements

This document describes the installation requirements and recommendations for IQLECT platform. If you have any question regarding compatibility and operation with your infrastructure, please contact us.

Desktop system requirements

When setting up your system, keep in mind that available memory is the most important factor affecting the size of data sets that you can load and analyze in IQLECT platform. Additionally, while there is no specific minimal requirement on the CPU, data analysis is a computationally intensive task — the better your hardware, the better your experience.

Configurations

  • Dual core
  • 64 GB RAM
  • Mimimum 1 TB free disk space
  • 8 Core minimum

Operating System

  • ubuntu 18.04
  • ubuntu 16.04
  • ubuntu 14.04
  • CentOS 7

Security Measures

  • Latest TLS Version
  • Data is stored in binary format.
  • Password Protected

Recommended

  • 16 core
  • ubuntu 18.04

Deployment Procedure

We need 2 builds : Portal build and DB build. Certain packages have to be installed. Nginx and other configurations needs to be changed. All this can be done by running a single deployment script.

Duration

Once the machines with required configurations are ready, deployment using the script takes around 1 hour on an average.
